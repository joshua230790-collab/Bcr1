Registro de Prédicas

Una app HTML lista para convertir a APK y guardar predicaciones, pastores, temas y el día del culto con un inicio de sesión elegante.

Inicio de sesión Usuario y contraseña con creación de cuentas dentro de la misma app.
Huella Botón preparado para el flujo biométrico. En HTML queda como desbloqueo local y luego puedes llevarlo a APK.
Registro Pastor, día, tema y el tipo de culto: lunes, miércoles, sábado o domingo.
Ordenado Lista de predicas guardadas, estadísticas básicas y eliminación individual.
Los datos se guardan en localStorage. Eso significa que quedan en el navegador o en el APK que construyas desde este HTML.

Iniciar sesión

Crea tu cuenta una vez y después entra normalmente o con huella local.

Huella: en HTML puro no se puede leer la huella real del sensor del teléfono. Este archivo deja listo el botón para el flujo biométrico y además guarda un acceso local para que luego lo lleves a APK con permiso nativo.
